# WHAT TO EAT TODAY?
현대인이 매일 직면하는 최대의 난제, "오늘 뭐 먹지?"  
What to eat today?는 오늘의 기분과 땡기는 맛, 날씨를 조합해 메뉴를 추천해주는 웹서비스 입니다. 

## 📄 FEATURES

+ 뉴모피즘 스타일의 UI 디자인
+ WEATHER API를 사용한 실시간 날씨 정보
+ 보안을 위한 Firebase 실시간 데이터 베이스 사용
+ SNS(카카오톡, 페이스북, 트위터) 공유 기능 구현
